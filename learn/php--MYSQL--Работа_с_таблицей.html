Работа с таблицей

ALTER

Команда для работы с таблицей

1. Переименование таблиц
ALTER TABLE classics RENAME pre1900;

2. Изменение типа данных столбца
ALTER TABLE classics MODIFY year SMALLINT;

Используется ключевое слово MODIFY
year SMALLINT - имя столбца и новый тип данных для этого столбца

3. Добавление нового столбца
ALTER TABLE classics ADD pages SMALLINT UNSIGNED;

ADD pages - добавляем новый стобец pages с типом данных SMALLINT UNSIGNED

4. Переименование столбца
ALTER TABLE classics CHANGE type category VARCHAR(16);

CHANGE type category - переименовываем столбец с именем type на category и указываем тип данных - VARCHAR(16)

5. Удаление столбца
ALTER TABLE classics DROP pages;

DROP pages - удаляем столбец pages
