console.time

Замер производительности;

Запускает таймер, который вы можете использовать для определения, сколько времени
занимает выполнение той или иной операции. Вы должны назначить каждому таймеру уникальное имя и
таким образом можете запустить до 10000 таймеров на одной странице.

Когда вы вызовете console.timeEnd() с именем, которое использовалось при запуске таймера, 
браузер отобразит в консоли время (в миллисекундах), прошедшее с момента запуска таймера.

console.time();
console.timeEnd();

console.time('first');

function sum(x, a) {
  return x + a;
}

console.log(sum(10, 7));

console.timeEnd('first');
