MYSQL

База данных

 - это структурированная коллекция записей или данных, хранящихся в компьютерной системе и
организованных так, что можно осуществлять быстрый поиск и извлечение нужной информации.
В названии MySQL составляющая SQL означает Structured Query Language - язык структурированных запросов.


Основные понятия:
• база данных — контейнер для всей коллекции данных MySQL;
• таблица — вложенный в базу данных контейнер, в котором хранятся сами данные;
• строка — отдельная запись, в которой могут содержаться несколько полей;
• столбец — имя поля внутри строки

Работа через терминал:
mysql -u root -p

-u - имя
-p - пароль

SHOW databases; - покажет все имеющиеся БД
