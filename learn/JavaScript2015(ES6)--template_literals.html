template literals

Строки-шаблоны (template literals)



Внимание!

Используются обратные кавычки:

const string = `Обратные кавычки`;



Вывод будет таким, каким была указана строка. Т.е. с сохранением пробелов, переносов и прочего.

const multiline = `Это
многострочная
фраза`;

console.info(multiline);

/*
"Это
многострочная
фраза"
*/



Конкатенация стала красивее!

`${...}`

const scripts = 3;
const styles = 2;
const template = `Скриптов — ${scripts}; \
                  cтилей — ${styles}; \
                  итого: ${scripts + styles}.`;


console.info(template); // Скриптов — 3; cтилей — 2; итого: 5.
