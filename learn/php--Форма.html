Форма

Суперглобальные переменные

$_POST
$_GET

Приходят из Формы

- Изначально это пустые массивы
- При отправке формы тем или иным методом, соответсвующий массивы заполняется данными,
которые есть в форме (name and value)

Так же post / get - являются методами отправки данных формы


Так можно посмотреть, какие данные были переданы в форму:
==
echo '<pre>'.print_r($_GET,1).print_r($_POST,1).'</pre>';
===


Пример формы:
<form action="" method="post">
      <p>Имя:<br>
        <input name="name" type="text">
      </p>

      <p>Фамилия:<br>
        <input name="surname" type="text">
      </p>

      <p>E-mail:<br>
        <input name="email" type="text">
      </p>

      <p>Сообщение:<br>
        <textarea name="message" cols="30" rows="5"></textarea>
      </p>

      <p><button type='submit'>Отправить</button></p>
</form>



На выходе мы получил массив $_POST с такими данными:
Array
(
    [name] => Имя
    [surname] => Фамилия
    [email] => email
    [message] => Сообщение
)


Обратиться к любому элементу можно, например: $_POST['name'];


- <input type="checkbox" name="question[]"> - если нужно передать несколько значений из поля чекбокс, то можно отправлять
массив со значениями;
На выходе получим массив question с "0" => *value*


Важно!
Все приходящие данные требуют проверки.
Без проверки велик шанс - что образуется уязвимость.

В примере выше, проверка может выглядеть так:

if (isset($_POST['name'], $_POST['pass'],)) {
    // Сделать что-то
}


Это касается обязательных данных. Необязательные можно проверять внутри условия на проверку обязательных;
